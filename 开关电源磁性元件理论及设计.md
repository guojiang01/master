# 开关电源磁性元件理论及设计 

1. 与磁化特性曲线相关的参数

   1. 饱和磁通密度：用足够大的磁场强度磁化磁性物质，当磁化曲线接近水平时，不再随外磁场增大而明显增大，对于高磁导率的软磁材料对应的B，即为饱和磁通密度。

   2. 剩余磁感应强度：铁磁物质磁化到饱和后，又将磁场强度下降到零时，铁磁物质中残留的磁感应强度，称为剩余磁感应强度

   3. 矫顽磁力：铁磁物质磁化到饱和后，由于磁滞现象，要使磁介质中的B为零，需有一定的反向磁场强度H，此磁场强度称为矫顽磁力。

      如果磁滞回线很宽，即$H_c$很高，需要很大的磁场强度才能将磁材料磁化到饱和，同时需要很大的反向磁场强度才能将材料中磁感应强度下降到零，也就是说这类材料磁化困难，去磁也困难，称这类材料为硬磁材料。如铝镍钴、钐钴和钕铁硼合金等永久磁铁，常用于电机激磁和仪表产生恒定磁场。这类材料磁滞回线宽，矫顽力高

      另一类材料在较弱外磁力磁场作用下，磁感应强度达到很高的数值，同时具有很低的矫顽力，即容易磁化，又容易退磁，称这类材料为软磁材料。属于这类材料的有电工纯铁、电工硅钢、铁镍软磁合金、铁钴钒软磁合金和软磁铁氧体等。开关电源主要应用软磁材料。某些特殊磁性材料，如恒导磁合金和非晶态合金也是软磁材料。

2. 磁导率：在磁芯的基本磁化曲线上，磁通密度B和磁场强度H之比称为磁芯的磁导率$\mu$。
   1. 绝对磁导率：$\mu=B/H$，得到材料的磁导率，也称为绝对磁导率。
   2. 相对磁导率：为了比较介质导磁性能，通常以真空磁导率为基准，定义介质的磁导率（与真空磁导率$\mu_0$之比为相对磁导率）$\mu_r$，即$\mu_r=\mu/\mu_0$。
   3. 最大磁导率：在某一磁场强度下，相对磁导率达到最大值，称为最大磁导率$\mu_m$。
   4. 初始磁导率：初始磁导率是磁性材料处于中性状态下磁导率的极限值，是磁元件的一种重要参数。一般规定材料样件是环形的闭合磁路，当激励磁场强度$H\rightarrow 0$ 时的磁导率称为初始磁导率$\mu_i$,$\mu_i=\frac{1}{\mu_0}\lim\limits_{H\rightarrow 0}\frac BH$。初始磁导率与温度和频率有关。
   5. 增量磁导率：在一个直流稳定磁场的作用下，叠加一个较小的交流磁场时，交流励磁分量沿局部磁滞回线变化，这时的磁导率称为增量磁导率。如果交流分量和直流分量比较，小到可以忽略，则增量磁导率也可以称为可逆磁导率。该值与直流磁场大小，磁芯的几何形状及温度有关。
   6. 有效磁导率：电感磁芯除了采用低磁导率环形磁芯或磁粉芯材料外，有时还采用开有空气隙的高磁导率磁芯。因此，常用有效磁导率表示开有气隙的磁芯的磁导率。高磁导率存储能量很少，主要用空气隙存储能量。
   7. 幅值磁导率：没有直流偏置时，交变磁场强度的幅值与磁通密度幅值的关系称为幅值磁导率，即$\mu_a=\frac{1}{\mu_0}\frac{B_p}{H_P}$。
   8. 复数磁导率：当铁磁体内存在磁场时，磁体被磁化。对于不同的材料，其磁化的难以程度不同，通常用磁导率$\mu$来表示材料的这种性质。在稳恒磁场中，材料的磁导率是实数。在交变磁场中，由于磁感应强度B落后与磁场强度H的变化，即B与H存在相位差，所以磁导率要用复数磁导率表示。引入复数磁导率的好处是，可以同时反应B与H间的振幅和相位关系。磁场强度H和磁感应强度B用复数表示为$\widetilde{H}=H_me^{j\omega t}$ $\widetilde{B}=B_me^{j(\omega t-\delta)}$ 。

3. 与铁芯损耗相关的参数

   1. 损耗因数：铁磁物质在交流磁化过程中，因消耗能量发热，磁性材料的损耗功率$(P)$由磁滞损耗$(P_h)$、涡流损耗$(P_e)$和剩余损耗$(P_r)$组成，即$P=P_h+P_e+P_r$。

      正是由于这三部分损耗，电感模型中由电阻分量，从而引起了电感电流和电压的相移（相位差，也称损耗角）。这三部分损耗所引起的相移可以表示为$tan\delta=tan\delta_h+tan\delta_e+tan\delta_r$，分别为损耗因数、磁滞损耗因数、涡流损耗因数和剩余损耗因数。

   2. 比损耗因数：磁芯磁路中通常存在气隙，使磁芯的有效磁导率小于材料的磁导率。把磁芯损耗表示为与材料特性有关而与磁芯磁导率无关的参数，更具有代表性，称为比损耗因数$tan\delta/\mu_i$。当磁芯磁路中存在气隙时，比损耗因数为$tan\delta/\mu_e$。有效磁导率和初始磁导率通常远大于1.
   3. 品质因数：当电感用于滤波器时，常用品质因数来表示其性能，品质因数定义为$Q=1/tan\delta=\omega L/R_{total}$，总电阻$R_{total}$包括某工作频率时绕组的等效电阻$R_{Cu}$。
   4. 磁滞损耗常数：当磁芯工作频率一定，磁通密度增加时，磁滞损耗增大。磁滞损耗占磁芯损耗的比例可通过测量在某一工作频率下。

4. 居里温度和磁导率的比温度系数
   1. 居里温度：随着温度升高，磁芯的磁导率发生变化，当相对磁导率突然变化到1时，该点所对应的温度问居里温度
   2. 磁导率的温度系数和比温度系数

5. 磁导率的时间减落因数：某些铁磁材料在经过动态磁中性化后，其起始磁导率随时间而降低，最后达到稳定状态。这种现象称为磁导率的减落，它是磁后效的一种表现形式。

6. 功率损耗密度：当磁性材料用作共模电感或信号变压器等弱磁信号磁性器件的磁芯时，常用相对损耗因数和品质因数来衡量其磁性能；当磁芯材料工作在高磁通密度并作为功率磁芯材料时，就用磁芯功率损耗密度来衡量其磁性能。

7. 磁芯损耗
   1. 磁化能量和磁滞损耗：磁性材料磁化时，送到磁场的能量包含两部分：一部分转化为势能，即去掉外磁化电流时，磁场能量可以返回电路；而另一部分变为克服摩擦使磁芯发热消耗掉，这就是磁滞损耗。
   2. 涡流损耗：因磁芯材料的电阻率不是无限大，绕着磁芯周边有一定的电阻值，感应电压产生电流，即涡流，流过这个电阻，引起损耗，这就是涡流损耗。涡流一方面产生磁芯损耗，另一方面产生的涡流所建立的磁通阻止磁芯中主磁通变化，使得磁通趋向于磁芯的表面，导致磁芯有效面积减小，这种现象称为集肤效应，类似于高频电流通过导体产生的集肤效应。通常定义为电流密度减小到导体截面表层电流密度的$1/e(e=2.718)$处的深度叫做集肤深度$\Delta$，单位m，通常可表示为$\Delta = \sqrt{\frac{\rho}{\pi\mu_0\mu_rf}}$，$\rho$为磁性材料的电阻率$\Omega\cdot m$，$\mu_r$为磁性材料的相对磁导率，$\mu_0=4\pi\times10^{-7}H/m$为空气的磁导率，$f$为磁通变化率Hz。
   3. 剩余损耗：是由于磁化弛豫效应或磁性滞后效应引起的损耗。$P_c=fB_m^{1.6}V$，高频时，涡流损耗和剩余损耗超过了磁滞损耗，磁芯损耗可表示为$P_c=\eta f^\alpha B_m^\beta V$，$\eta$为损耗系数，f为工作频率，$B_m$为磁芯幅值磁感应强度，V为磁芯体积。

8. 动态磁化

   1. 磁化过程是在直流或低频磁场的情况下进行的，磁化过程中的滞后现象即弛豫现象可以忽略。磁性材料从一个稳定磁化状态转变到新的平衡状态，如不考虑建立新的平衡过程的时间问题，可以称为静态磁化过程。在静态磁化过程中也会因不可逆磁化出现磁滞现象，但其每个磁化状态都处于亚稳定状态，并且磁化状态不随时间改变。许多磁性材料，如硅钢片、坡莫合金、镍锌铁氧体等，需要在交变磁场中使用，因此需要考虑磁化的时间问题。

      磁性材料在周期性变化的交变磁场中时，其磁化强度也周期性地反复变化，构成动态磁滞回线。在相同的磁场强度范围内，动态磁滞回线的面积比静态磁滞回线要大些。这是因为磁滞回线的面积等于磁化一周所消耗的能量。在静态磁场下，材料内的损耗仅为磁滞损耗，而在交变磁场下，材料内除了磁滞损耗外还有物流损耗和剩余损耗等。

   2. 高频下的磁化曲线：当交变磁场强度减小或增加交变磁场频率时，动态磁滞回线的形状将逐渐趋近于椭圆。

      磁化落后磁场变化的现象称为时间效应/磁化的时间效应表现为一下集中现象：

      1. 磁滞现象
      2. 涡流效应
      3. 磁导率的频散和吸收现象
      4. 磁后效

9. 软磁材料的重要指标：
   1. 初始磁导率：高初始磁导率是软磁材料的基本要求，理论和实践证明，降低软磁材料的杂志浓度，提高密度，增大晶粒尺寸，结构均匀化，降低磁致伸缩系数，消除内应力和气孔的影响是提供初始磁导率的充分条件。
   2. 要求具有很小的矫顽力和狭窄的磁滞回线。$10^{-1} - 10^2 A/m$。
   3. 电阻率高
   4. 具有较高的饱和磁感应强度
   5. 磁芯损耗
   6. 稳定性

11. 合金磁材料：合金材料是几种金属材料熔化冶炼而成一定的金相结构的材料，是由基本磁性材料铁、镍、钴或加入其他元素构成的合金。除恒导合金外，合金材料通常有极高的相对磁导率(60000)，很高的饱和磁通密度(0.6~1.9T)、接近饱和磁通密度的剩余磁感应强度和很窄的磁化特性曲线。特别是铁镍或铁镍钼合金，低频磁化特性曲线很接近理想矩形磁化曲线。因其磁芯存储能量很少，非常适宜做变压器、磁放大器、尖峰抑制器、滤波电感、有缘PFC电感等器件的磁性材料。

    金属带状或叠片状软磁合金材料主要有坡莫合金、超坡莫合金、非晶合金、超微晶合金、硅钢、钴铁合金和超钴铁合金。合金材料的缺点是电阻率非常低，为了减少涡流效应，这类合金磁材料都是碾轧成带料卷绕而成。

    1. 硅钢片：在电工纯铁中加入少量硅，形成固溶体，可提高合金的电阻率，减少材料的涡流损耗。并且随着纯铁中含硅量的增加，磁滞损耗降低，弱磁场和中等磁场下，磁导率增加。电工硅钢片的制造工艺分为热轧和冷轧两种，以结晶温度为区分点。冷轧一般用于生产带料，轧速高，避免摩擦带来的温度升高。冷轧钢沿着碾轧方向的磁感应强度最大，损耗小，具有各向异性，加工难度大。热轧钢沿着碾轧方向的磁感应强度小，损耗大，具有各向同性，加工容易。

    2. 坡莫合金磁性材料：镍含量30%~90%的铁镍系合金。坡莫合金具有极高的初始磁导率(超过1000000/H/m)、极低的矫顽力(小于0.002Oe)和磁化曲线高矩形比的软磁材料。

12. 非晶合金和微晶合金

    1. 非晶合金的种类：
       1. 铁基非晶合金：其特点是磁性强(饱和磁通密度可达1.4~1.7T)、磁导率、激磁电流和铁损耗等软磁性能优于硅钢片，价格便宜，最适合替代硅钢片，特别是铁损耗低，代替硅钢片做配电变压器可降低铁损60%-70%。
       2. 钴基非晶合金：价格贵，磁性较弱（1T以下），但磁导率极其高，矫顽力极低，适用于几十到几百千赫兹
       3. 铁镍基非晶合金：中等的饱和磁感应强度（0.7-1.2T）、较低的磁芯损耗和很高的磁导率
       4. 纳米晶软磁合金材料或超微晶软磁合金材料：高的初始磁导率($10^5H/m$)、高饱和磁通密度(1.2T)、低比损耗$(P_{0.2/50kg}=15W/kg)$及良好的温度稳定性。20kHz以上，数百kHz以下

    2. 非晶合金的特点：
       1. 优良的磁性能
       2. 灵活的处理工艺
       3. 制造工艺简单、节能和环保
       4. 机械强度较高且硬度较硬

13. 金属磁粉芯磁芯材料：磁粉芯通常将磁性材料极细(0.5~5$\mu m$)的粉末和粘接剂混合在一起，通过模压、固化形成粉末金属磁芯。由于磁粉芯中存在大量的非磁物质，相当于在磁芯中存在许多非磁分布气隙，磁化时，这些分布气隙中要存储相当大的能量，因此可用这种磁芯作为直流滤波电感和反激变压器磁芯。

       4. 铁硅合金磁粉芯：饱和磁通密度1.6T，居里温度700度
       5. 铁粉芯：初始磁导率低，饱和磁通密度1.6T，价格低，损耗大，大概是铁氧体损耗的65倍。

14. 磁粉芯一般比铁氧体有更高的饱和磁通密度，用磁粉芯的电感比铁氧体磁芯的体积小，但在100KHz以上应用时，损耗大，很少再用磁粉芯。铁氧体价格低廉，材质的磁芯价格齐全，高频性能好，损耗大，很少再用磁粉芯。铁氧体价格低廉，材质和种类齐全，高频性能好，但材质脆，不耐冲击，温度性能差，适用于10KW以下，最高频率大1MHz以下的任何功率变换器。

    在开关电源中常用的磁性材料有三类：合金材料、磁粉芯和铁氧体。磁粉芯和合金带料中恒导合金只用于电感或反激变换器磁芯，由于不用开气隙，散磁少，电磁兼容性好。但绕线困难，不适宜做大功率磁芯。合金带料一般具有高饱和磁通密度，低电阻率，损耗大。其中超微晶材料工作频率达200KHz，仍具有可接受损耗的较高工作磁通密度，并可作为大功率变压器。矩形慧娴非晶材料可作为磁放大器和尖峰抑制磁芯。铁氧体是开关电源中应用最为广泛的材料，价格便宜，磁芯价格齐全，最高工作频率可达1MHz以上。

15. 高频电流引起的集肤效应：

    从表面到电流密度$j$下降到表面电流密度的0.368（1/e）的厚度为趋肤深度或集肤深度$\Delta$，即认为导线表面下深度为$\Delta$的厚度内的导体中流过导线的全部电流，而在$\Delta$厚度外的导体内完全不留过电流。$\Delta$与频率$f(\omega)$和导线物理性能的关系为：$\Delta=\sqrt{\frac{2k}{\omega\mu\gamma}}$，$\mu $为导线材料的磁导率，$\gamma=1/\rho$为材料的电导率，k为材料电导率（或电阻率）的温度系数。

16. 交直流电阻比与线径和频率的关系：直流电阻反比于导线的截面积，因集肤效应使导线的有效截面积减小，交流电阻增加，当导线直径大于2倍集肤深度时，交直流电阻之比可表示为导线截面积与集肤面积之反比，$\frac{R_{ac}}{R_{dc}}=\frac{\pi d^2/4}{\pi d^2/4-\pi(d-2\Delta)^2/4}=\frac{(d/2\Delta)^2}{(d/2\Delta)^2-[d/(2\Delta)-1]^2}$，由于集肤效应，导线的交直流电阻比取决于导线径d和集肤深度$\Delta$的比值。开关电源拓扑中，电流为矩形波，包含大量的谐波能量，估计交流与直流嗲组做法是将开关频率的前3次谐波集肤深度取平均值，再由平均值求比值。粗略计算时，矩形波电流集肤深度为基波正弦波集肤深度的70%。

17. 邻近效应：将各线圈分成两段，按N1/2-N2/2-N1/2-N2/2交错绕制，各段线圈处于1/2总磁场强度中，邻近效应减小。原边和副边线圈位置可以互换。分段越多，每段层数越少，但绝缘越多，窗口填充系数越低。每个线圈一般不超过两段。线圈更合理的平分方式是原边或副边之一分成两段，而另一个不分段，被两个1/2线圈夹在中间，称为三明治绕法。

    1. 邻近效应引起的被动损耗：处于交变磁场中不通电导体的损耗称为被动损耗

       + 无源导体的邻近效应“如果导体位于原、副边之间的高磁场强度区，即使导体不是线圈的一部分或不处在工作时间也会引起损耗。这种情况包括：线圈间电磁屏蔽，轻载或空载的副边线圈，如中心抽头，暂不通电流的线圈以及处于散磁区的线圈等。
       + 表面电流可能十分大，引起了明显的附加线圈损耗，减少和限制无源线圈损耗的措施有：
         1. 将线圈放置到高交流磁场区外
         2. 通过交错和采用宽窗口的磁芯，减小磁场强度
         3. 采用更薄导体

       + 法拉第屏蔽避免了原、副边之间的耦合。而屏蔽总是处于最高磁场强度。因为屏蔽层电流很小，导体厚度可能并应当远小于肌肤深度。
       + 尽量避免中心抽头线圈：在有中心抽头线圈中，一边不工作，而另一边是导通的。这不仅窗口利用率不好，而且不工作线圈常位于工作边与反向磁场线圈之间高磁场区，因而承受无源损耗。
       + 避免原边中心抽头线圈并不困难，可选择正激、桥式或半桥拓扑。对于反激变换器，原边和副边线圈不是同时有电流，为了避免邻近效应，通常尽量减少层数，采用比一般推荐的要高的电流密度选择导线尺寸。
       + 减少散磁通：高频磁芯线圈的散磁会带来以下的严重后果：
         1. 散磁通引起周围电路的电磁干扰
         2. 散磁通引起周围电路损耗
         3. 散磁通引起铜箔线圈导体涡流，减少导体有效截面积，增加导体损耗，或引起导体局部过热

20. 线圈间电容和端部电容

    1. 线圈间电容是开关变压器原边到副边之间共模噪声的通道。在减少漏感和涡流损耗时，要求线圈交错绕，宽窗口，原、副边空间尽量紧凑等都增加了线圈间电容

    2. 为了减少原边和副边之间的电耦合，构成电气屏蔽。屏蔽层必须本身绝缘，不能构成电路匝。因为屏蔽层总是处于高磁场强度区，屏蔽层厚度必须远小于肌肤深度，一般为三分之一，避免被动涡流损耗。屏蔽层应当以最小的引线电感直接焊接到变压器原边线圈的电压不变端或大地，才能起到屏蔽作用。

    3. 端部之间电容：也称为分布电容。在高压线圈中，线圈来回绕城许多层。上一层的末尾与下一层的开头之间存在电容，交流电位差大，形成旁路电流。在变压器中，原边分布电容与互感和漏感引起串联和并联谐振。副边在二极管反向恢复时与漏感引起振荡和尖刺。在滤波电感中，端部电容及上述谐振，将通过开关电源的高频分量传输到输出。在低压和低阻抗线圈中可以忽略端部电容，一般单层线段端部电容很小。在射频滤波阻流圈的多层线圈中，采用按磁芯长度方向分段绕，或采用蜂房式绕法，端部电容可大大减少。

22. 高频变压器设计的基本问题

    1. 几种常见的磁芯材料结构
       1. 环形磁芯：最大的问题是绕线困难，而且在匝数少的情况下无法均匀地绕在磁芯上，用铁氧体打造磁芯，主要用于脉冲变压器和宽带变压器和EMC滤波电感，很少用于开关电源的功率变压器，行业标准规定内外径之比大于0.6.
       2. E、EE、EC、ETD型磁芯：相对于外形尺寸来说具有较大的窗口面积、较宽窗口宽度的结构，加大的线圈宽度可减小线圈的层数，使交流电阻和漏感减小。固定的爬电距离对宽窗口影响较小，宽窗口需要的线圈高度低，因此可以更好地利用线圈窗口面积。开放式的窗口没有出线问题，线圈和外界空气接触面积大，有利于空气流通散热，可处理大功率，但EMI大。
       3. 平面E型磁芯：铜损耗和磁芯损耗发热的传热路径很短，散热面积大，适合做高功率密度的产品。
       4. 罐型磁芯(PQ、RM、PM):较小的窗口面积，窗口形状几乎是方形的。有较好的磁屏蔽的优点，减少了EMI。但是这类磁芯的爬电尺寸耗费了窗口较大的面积，窗口宽度有限，只能用于125W以下的低功率场合，线圈的散热十分困难，不适合大功率场合，引出线缺口小，不适宜多路输出。
       5. RM型磁芯：比罐型更大的出现窗口和好的散热，因而可传输更大的功率。
       6. UU型和UI型磁芯：主要用在高压和大功率场合，很少用于1kw以下，可以用更粗的导线和更多的匝数，漏感更大。
       7. 其他类型的E型磁芯：适合做电源变压器和输出滤波电感。
    2. 最大磁通密度的选择：$N_1=\frac{UT_{on}}{A_c\Delta B}$，$U$为原边绕组上的电压，$A_e$为磁芯的有效截面积，$\Delta B$磁通密度摆幅，最大磁通密度$B_m$越大，磁通密度摆幅$\Delta B$也越大，原边线圈$N_1$的匝数越少，可允许的绕线尺寸就越大，因此磁芯处理功率的能力越大。多数铁氧体在频率小于25KHz时磁芯损耗都很小，在这种频率下最大磁通密度可以不考虑磁芯损耗的影响。当频率大于50KHz时，必须采用低损耗的磁芯材料，或者适当减小最大磁通密度。大部分铁氧体材料磁滞回线上线性部分的最大值约为2000Gs，如果超过这个范围，功率开关管导通阶段结束的时刻励磁电流将会增大，进而增加线圈损耗和开关管导通损耗。对大多数铁氧体来说，在计算原边匝数的适合取最大磁通密度为2000Gs是由风险的，因为当电网电压或负载跳变时，如果反馈误差放大器响应速度不够快，在几个开关周期内无法调节过来，那么最大磁通密度将会达到饱和值并且损耗功率开关管。因此最大磁通密度选择为1600Gs左右。

    4. 直流偏磁
       1. 直流磁化：在磁化特性曲线上可以看到，由于磁芯存在矫顽力为$\pm H_c$，当磁场反复对磁芯进行磁化时，总需要额外地有一部分能量用来克服矫顽力和消除剩余磁通，这一部分用来克服矫顽力和消除剩余磁通的磁场能量，对于磁芯来说，是不起增强磁通密度作用的，且磁感应强度的变化总是要落后于磁场强度一个相位。励磁电流或磁场强度从最大值下降到零，但磁通密度却不是跟随磁场强度下降到零，而是停留在一个被称为“剩磁”的剩余磁通密度位置上，如果此时再施加激磁电流，磁芯将沿新的曲线磁化，这样几个循环后，磁芯就会饱和。由于磁芯不能复位，出现向一个方向缓慢磁化直至饱和的现象称为直流偏磁。
       2. 直流脉冲对磁芯的磁化： 每输入一个直流脉冲电压，变压器磁芯中的磁通密度就要线性增长和下降一次，在开始输入直流脉冲电压的时候，磁通密度增长的幅度大于下降的幅度。开始时磁场强度对变压器磁芯进行磁化时还没有使磁通密度或磁矫顽力达到接近饱和的程度；要经过若干个过程后，磁通密度增长的幅度与下降的幅度才会一样大，这说明变压器磁芯中的矫顽力以及基本达到饱和。当变压器磁芯初次被直流脉冲电压产生的磁场磁化的时候，磁场强度和励磁电流的变化幅度都要经过一个过程过渡，然后才能基本趋于稳定，并且磁场强度和励磁电流变化的幅度由小到大，是因为变压器磁芯开始的时候磁导率比较大，然后磁导率逐步变小的缘故。
       3. 交变磁化：低频率变压器中，原边线圈的电阻压降足以限制磁芯趋于饱和，这是因为当小的直流分量将磁通慢慢推向饱和时，磁化电流开始不对称。直流分量增加的磁化电流在线圈电阻上产生一个IR降落，抵消了激励波形中的直流电压分量，有可能避免磁芯饱和。
       4. 磁通不平衡的处理方法：
          1. 磁芯加气隙：当磁芯偏离磁滞回线的弯曲部分（铁氧体磁通密度大于2000Gs）时，磁通不平衡就变得很严重了，励磁电流开始按指数规律增加。如果改变磁滞回线倾斜度吗，使磁滞回线的弯曲部分可以延申到更大的电流区域，可以减小磁通不平衡现象的发生，使磁芯可以承受更大的直流电流偏置或伏秒数不平衡。
          2. 原边串联电阻：当伏秒数不平衡时，原边线圈电阻可阻止磁芯迅速饱和。当出现磁通不平衡时，伏秒数较大的原边半绕组流过较大的峰值电流。这个电流使绕组压降增大，降低了原边半绕组的伏秒数，从而使电流恢复平衡。

23. 电感设计：为了达到减少匝数和铜损耗的要求，最理想的磁芯材料应该具有高的磁导率和小的磁损耗。但在电感设计中，大直流电流经过电感元件及实际中所用磁性材料的有限饱和磁通密度，不得不选用低磁导率材料和磁芯中引入气隙。然而，由于过低的有效磁导率，需要更多的匝数绕组来达到所需的电感值。
    1. 磁粉芯材料电感设计：由于气隙分散，有效降低了初始磁导率。通常初始磁导率在$10-550\mu$之间的磁芯材料可用来设计电感。
       1. 磁粉芯材料的饱和特性：通常的材料有间隙铁氧体、铁粉芯、MPP和Kool Mu等材料。间隙铁氧体并没有因为气隙而改变饱和磁通密度，只是改变了饱和时的磁场强度。其他条件一样，饱和磁通密度高的材料，电感量更大，铜耗更低。
       2. 低交流应力时限制因素：当交流损耗或工作频率低时，磁芯损耗不是主要限制因素，所有可以选磁导率较高的磁粉芯材料。这种材料的特点是饱和磁通密度高，磁导率高且成本低。采用磁导率较高的磁芯，只需要较少的匝数就可以得到所需的电感量，从而可降低铜耗。磁芯的磁通密度较高意味着在较大的直流偏置电流下磁芯仍然不饱和。频率和纹波电流都很低时，则磁芯损耗不严重，铜耗远超过磁芯损耗。

24. 特殊磁元件设计

    1. 尖峰噪声的抑制
       1. 矩形磁性材料的基本特性：如果环的内外径比小，这种材料环的磁化曲线将发生倾斜。如果采用矩形磁滞回线的磁性材料构成电感，这种电感称为饱和电感，在电子电路中常被当做可控延时开关元件使用。
       2. 抑制尖峰原理：非晶磁珠是一个环形磁芯的单匝电感，把它穿在二极管的引线上作为一匝可饱和电感，用来抑制二极管反向恢复电流。

    2. 磁放大器

    3. 电流互感器：电流的采样一般可用功率电阻、分流器和霍尔元件。用功率电阻采样电流损耗大，易受干扰，很难做到高精度测量；用分流器检测，适合比较小的功率场合且体积大；用霍尔元件虽然精度高，但成本太高。

       理想的电流互感器其负载阻抗应恒定，漏磁通为零，激磁电流为零，以及具有无限大的磁通密度。对检测互感器来说负载阻抗一般是恒定的，由于采用环形磁芯，副边又均匀分布在磁环的圆环 上，可以忽略的漏感是可以做到的。原边一般是1匝或很少几匝，副边匝数很多，原副边绕组匝比一般为1：100或者更多。

       1. 交流互感器：交流互感器一般采用环形磁芯，原边线圈N1为1匝或数匝，副边N2的匝数较多。为了便于测量，副边通常接有检测电阻R，将电流信号变换成电压信号。理想情况下互感器的激磁电感$L_1\rightarrow\infty$无穷大，激磁电流$\dot I_m=0$，则$\dot I_1=\dot I_2$,实际激磁电感不可能无穷大，总是存在激磁电流。为了维持$I_2$，副边感应电势为$e_2=I_2(R+R_{Cu})=4.44fBAN_2$，$R_{Cu}$为副边线圈电阻，f为电流频率，B为磁芯工作峰值磁感应强度，A为磁芯截面积，R为检测电阻。

          副边反射电流与原边电流的相位差$\alpha=\arctan\frac{R'}{\omega L_1}$，$R'=(R+R_{Cu})N_1^2/N_2^2$为副边反射到原边的阻抗；原边激磁电感$L_1=N_1^2\mu_0\mu_aA/l$。在一定的频率下，应当减少检测电阻或增加副边激磁电感。为了减少检测误差，增加副边激磁电感是提高检测精度的唯一的方法。

       2. 交流电流互感器设计：交流互感器设计前应当知道互感器的工作频率$f$，被检测的电流即原边电流$I_1$，副边所需电压$U_2$（有效值）和检测精度$\gamma$。互感器设计原则是要保证电流检测精度，初始设计时可不考虑线圈电阻$R_{Cu}$，在副边激磁电抗远远大于检测电阻时，$\alpha=R/(\omega L_2)$，$N_2A_L=\frac{U_2}{\omega  I_1\sqrt{2\gamma}}$ 。根据给定允许的幅值误差$\gamma$或允许的相位误差$\alpha$，选择磁芯尺寸和副边线圈匝数。选取较多的副边匝数对提高测量精度是有益的。但是副边匝数过多，一方面绕线困难，另一方面导线长度增长，线圈电阻增加，又降低了检测精度，一般N2在500匝以下。

          根据工作频率选择磁芯材料，例如50Hz选用硅钢片厚0.35mm的环形磁芯，400Hz选用0.1mm高硅薄带环形磁芯，高于10KHz选用非晶态或铁氧体材料等，选择尽量高$\mu$的材料。如果要求检测相位误差极小低频是应选择$\mu$极高的坡莫合金或非晶态磁芯。还应当注意，$\mu_a$为幅值相对磁导率，在低磁感应强度时一般和初始磁导率$\mu_i$相近，初始设计时，可用$\mu_i$代替$\mu_a$。

       3. 影响检测精度的因素分析：交流互感器在副边接有电阻时，原边电流中只有很小一部分，即检测电流的百分之几用来磁化磁芯。为了提高检测精度，磁芯中磁感应密度远低于饱和磁感应密度。

          1. 二次侧线圈内阻$R_2$和漏抗$X_2$对误差的影响：当电阻$R_2$增大时，幅值误差和相位误差都增大，$X_2$增大时幅值误差增大，但相位误差减小。因此要改善误差应尽量减小$R_2$和适当的$X_2$值。对小容量的电流互感器影响才较显著。
          2. 线圈匝数对误差的影响：增加线圈匝数就是增加安匝数，可以使磁通密度减小，其改善误差的效果比增加铁芯面积显著。但是线圈匝数的增加会引起用铜量的增加，同时引起稳定倍数的减小和饱和倍数的增加。此外，如穿心型或套管型电流互感器一次侧线圈只有1匝，不能用增加匝数的办法改善误差，
          3. 减小铁芯损耗和提高磁导率：在铁芯磁通密度不变的条件下，减少铁芯励磁安匝和损耗安匝也将改善幅值误差和相位误差，因此采用优质的磁性材料和采取适宜的退火工艺都能达到提高磁导率和减小损耗的目的。铁芯磁性的优劣还影响饱和倍数，铁芯磁性差时饱和倍数较小。
          4. 工作频率较高时的误差分析：当工作频率高于检测频率时，相位差减小，检测误差也随之减小。因此只要基波频率满足误差要求，高频误差是很小的，但高频是应注意磁芯损耗和分布电容的影响。
          5. 运行中的电流互感器的误差：当电力互感器已经定型，其内部参数已经确定，互感器的误差将受到二次侧电流或一次侧电流、二次侧负载、功率因数及频率的影响。
             + 电流频率的变化对误差的影响：频率变动不但影响铁芯损耗、磁通密度和线圈漏抗的大小，也同时影响了二次侧负载电抗值
             + 一次侧电流减小的影响：当一次电流减小时，磁通密度按比例相应减小，但在低磁通密度时，励磁安匝的减少比磁通密度减少要慢，因此幅值误差和相位误差的绝对值就相对大。
             + 电流互感器的误差特征：当一次侧电流在规定的范围内变化时，二次电流按比例变化，当二次负载阻抗在规定范围内变化时，不影响二次侧电流的大小。所以当二次侧负载在额定范围内减少时，磁通密度也减少，由于二次侧电流不变，励磁电流减小，误差也将减小。
             + 二次侧负载的功率因数的影响：功率因数增大，也就是电阻R增大，阻抗X减小，相位误差将增大而幅值误差将减小。

    4. 脉冲直流互感器：脉冲直流互感器与交流互感器不同，交流信号使磁芯双向对称磁化，而直流脉冲互感器使磁芯单向磁化，属于正激变换器工作方式。
